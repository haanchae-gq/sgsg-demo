# 세션 5: 리뷰 API

## 세션 개요
이 세션은 서비스 이용 후 고객이 전문가에 대한 리뷰를 작성, 조회, 수정, 삭제할 수 있는 기능을 구현합니다. 리뷰 목록 조회, 상세 조회, 도움됨 표시 기능을 포함하며, 전문가 평점 계산과 연동됩니다.

**우선순위**: P2 (주문/결제 후)  
**예상 작업 시간**: 2-3일  
**의존성**: 세션 1 (인증), 세션 4 (주문 관리)  
**다중 에이전트 작업 가능**: 예 (단일 에이전트로 충분)

## 구현할 API 엔드포인트

### 리뷰 API (`/api/v1/reviews`)
| 메서드 | 엔드포인트 | 설명 | 권한 | 구현 상태 |
|--------|------------|------|------|-----------|
| POST | `/reviews` | 리뷰 작성 (주문 완료 후) | Authenticated (고객만) | ⚠️ TODO 필요 |
| GET | `/reviews` | 리뷰 목록 조회 (필터링) | Public | ⚠️ TODO 필요 |
| GET | `/reviews/:reviewId` | 리뷰 상세 정보 | Public | ⚠️ TODO 필요 |
| PUT | `/reviews/:reviewId` | 리뷰 수정 | Authenticated (작성자만) | ⚠️ TODO 필요 |
| DELETE | `/reviews/:reviewId` | 리뷰 삭제 | Authenticated (작성자만) | ⚠️ TODO 필요 |
| POST | `/reviews/:reviewId/helpful` | 도움됨 표시 | Authenticated | ⚠️ TODO 필요 |

## 필요한 데이터베이스 테이블/마이그레이션
### 이미 존재하는 테이블 (Prisma schema 확인)
- `Review` - 리뷰 기본 정보
- `Customer` - 고객 정보 (참조)
- `Expert` - 전문가 정보 (참조)
- `Order` - 주문 정보 (참조)

### 필요한 마이그레이션
1. `Review` 테이블 확장: 도움됨 카운트, 신고 상태, 관리자 메모 필드 추가
2. 인덱스: 전문가ID별 리뷰 조회 성능을 위한 인덱스
3. 전문가 평점 계산: `Expert` 테이블에 평점 관련 필드 확인/추가

## 의존성
### 내부 의존성
- **세션 1 (인증)**: 사용자 인증, 고객 역할 검증
- **세션 4 (주문 관리)**: 주문 완료 상태 확인 (리뷰 작성 조건)
- **세션 2 (전문가 관리)**: 전문가 평점 업데이트

### 외부 의존성
- 없음 (모든 데이터 내부 처리)

## 테스트 시나리오
### 단위 테스트
1. `ReviewService` 단위 테스트: 리뷰 생성, 조회, 수정, 삭제
2. `ReviewValidationService` 단위 테스트: 리뷰 작성 조건 검증 (주문 완료 여부)
3. `HelpfulService` 단위 테스트: 도움됨 표시 기능
4. `ExpertRatingService` 단위 테스트: 전문가 평점 계산 및 업데이트

### 통합 테스트 (E2E)
1. 주문 완료 → 리뷰 작성 → 전문가 평점 반영 전체 흐름
2. 리뷰 목록 조회 및 필터링 (전문가별, 평점별, 기간별)
3. 리뷰 수정 및 삭제 권한 검증
4. 도움됨 표시 및 취소 기능

## 다중 에이전트 작업을 위한 특별 지침
1. **작업 분할**: 
   - 단일 에이전트 권장 (규모가 작고 로직이 연속적)
   - 필요시: 에이전트 1(리뷰 CRUD), 에이전트 2(평점 계산/도움됨 기능)
   
2. **공유 리소스**:
   - `src/routes/v1/reviews/`: 라우트 구조
   - `src/services/review.service.ts`: 서비스 로직
   - 전문가 평점 계산 로직: `expert.service.ts`와 연동

3. **통합 포인트**:
   - 주문 상태 검증: 리뷰 작성 전 주문 완료 여부 확인
   - 전문가 평점 실시간 업데이트: 리뷰 작성/수정/삭제 시 반영
   - 데이터 정합성: 리뷰 삭제 시 평점 재계산

4. **비즈니스 규칙**:
   - 리뷰 작성 조건: 주문 상태가 'completed'이고, 주문 완료 후 30일 이내
   - 리뷰 수정 조건: 작성 후 7일 이내만 수정 가능
   - 도움됨 표시: 동일 사용자 중복 표시 방지

5. **테스트 협업**:
   - 세션 4(주문) 팀과 테스트 데이터 협의
   - 전문가 평점 계산 정확도 검증
