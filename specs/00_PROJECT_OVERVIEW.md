## 프로젝트 개요

### 서비스 소개
**쓱싹(SsgSak)**은 홈케어 서비스(설치/시공, 클리닝, 막힘해결)를 제공하는 전문가-고객 매칭 플랫폼입니다.

### 비즈니스 목표
- 전문가와 고객을 효율적으로 연결
- 투명한 가격 책정 및 결제 시스템
- 신뢰할 수 있는 서비스 품질 관리
- 공정한 전문가 정산 시스템
- 마스터/서브 계정 시스템을 통한 팀 기반 운영 지원

### 핵심 사용자
1. **전문가**: 서비스를 제공하는 기술자/업체 (마스터/서브 계정 지원)
2. **운영팀**: 플랫폼을 관리하는 백오피스 사용자 (RBAC 기반 권한 관리)
3. **고객**: 서비스를 이용하는 일반 사용자 (향후 개발)

### 핵심 기능
1. **마스터/서브 계정 시스템**: 전문가 팀 운영을 위한 계정 계층 구조
2. **3단계 서비스 카테고리 관리**: 대분류 → 중분류 → 서비스 항목 계층 구조
3. **가격 스냅샷 시스템**: 거래 내역 보호를 위한 주문 시점 가격 고정
4. **실시간 알림 시스템**: Aligo API 기반 KakaoTalk/SMS 통합
5. **정산 시스템**: 전문가별 주기적 정산 및 수수료 관리

## Spec-Driven Development 접근법

### 개발 원칙
1. **스펙 문서 우선**: 모든 구현은 스펙 문서 작성 후 시작
2. **단계적 검증**: 각 개발 단계별로 스펙 검증 및 테스트
3. **기술 스택 일관성**: Fastify + Prisma + Ant Design 통일
4. **독립적 개발 단계**: Phase 1-4 독립 개발 가능 (의존성 최소화)

### 문서 구조
총 10개 스펙 문서로 구성:
1. **00_PROJECT_OVERVIEW.md** (현재 문서) - 프로젝트 전반 개요
2. **01_ARCHITECTURE_DESIGN.md** - 시스템 아키텍처 및 컴포넌트 설계
3. **02_DATABASE_SCHEMA.md** - Prisma 스키마 상세 설명
4. **03_BACKEND_API_SPEC.md** - Fastify API 엔드포인트 상세 명세
5. **04_EXPERT_WEBAPP_SPEC.md** - 전문가 웹앱 기능 및 UI/UX 사양
6. **05_BACKOFFICE_SPEC.md** - 백오피스 관리자 기능 사양
7. **06_CONSUMER_WEB_SPEC.md** - 소비자 웹 기능 사양 (향후)
8. **07_EXTERNAL_SERVICES.md** - 외부 서비스 통합 사양
9. **08_DEPLOYMENT_STRATEGY.md** - 배포 및 운영 전략
10. **09_DEVELOPMENT_GUIDELINES.md** - 개발 가이드라인 및 표준

### 검증 프로세스
1. **스펙 작성** → 2. **기술 검토** → 3. **구현** → 4. **테스트** → 5. **문서 업데이트**

## 기술 스택 (v3.0)

### 백엔드 (Phase 1 - 최우선)
```yaml
언어: Node.js (v20+)
프레임워크: Fastify (Express 대체)
데이터베이스: PostgreSQL 16+
ORM: Prisma (Type-safe 데이터베이스 클라이언트)
인증: JWT (jsonwebtoken) + Fastify 플러그인
실시간: Socket.io (웹소켓 통신)
검증: @sinclair/typebox (런타임 타입 검증)
파일 업로드: @fastify/multipart
보안: @fastify/helmet, @fastify/cors, @fastify/rate-limit
```

### 프론트엔드 (Phase 2-4)
```yaml
라이브러리: React 18 + TypeScript
상태관리: Zustand (간결한 상태 관리)
라우팅: React Router v6
UI 프레임워크: Ant Design v5 (Tailwind CSS 대체)
  - 전문가 웹앱: 모바일 퍼스트, 하단 네비게이션 패턴
  - 백오피스: 데스크톱 최적화, 사이드바 레이아웃
HTTP 클라이언트: Axios (인터셉터 통합)
폼 관리: React Hook Form + Zod (유효성 검증)
테이블: Ant Design Table 컴포넌트
차트: Ant Design Charts / Recharts
```

### DevOps & 인프라
```yaml
컨테이너: Docker + Docker Compose
데이터베이스: PostgreSQL (메인), Redis (캐시/세션)
리버스 프록시: Nginx (SSL/TLS, 정적 파일 서빙)
CI/CD: GitHub Actions (자동화 빌드/테스트/배포)
모니터링: Winston (로깅), PM2 (프로세스 관리)
환경 관리: dotenv + 환경별 설정 파일
```

### 외부 서비스 (기존 계정 활용)
```yaml
결제: Hecto Financial (PG) - devsgsgcare / sgsgcare1!
SMS/카카오톡: Aligo API - sgsgcare / !@Caresg9581
이메일: SendGrid 또는 서버 내 Sendmail
파일 저장: AWS S3 (향후) 또는 로컬 스토리지 (초기)
```

## 개발 우선순위 및 단계

### 개발 철학
**"백엔드 안정성 → 관리자 효율성 → 전문가 편의성"** 순서로 개발

### Phase 1: 백엔드 API (4-5주) - 독립 개발 가능
**목표**: 안정적이고 확장 가능한 Fastify API 서버 구축

**핵심 기능**:
- 사용자 인증/인가 시스템 (JWT + RBAC)
- 마스터/서브 계정 관리 API
- 서비스 카탈로그 관리 API (3단계 계층)
- 주문 관리 시스템 (가격 스냅샷 포함)
- 결제 통합 (Hecto Financial PG)
- 정산 시스템 기본 구조
- 실시간 알림 웹훅 (Aligo 연동)
- 파일 업로드 처리

**출력물**:
- 완성된 Fastify API 서버 (`sgsg-api/`)
- Prisma 스키마 및 마이그레이션
- 전체 API 엔드포인트 문서화
- 통합 테스트 스위트

### Phase 2: 백오피스 관리자 (3-4주) - Phase 1 완료 후
**목표**: 운영팀이 플랫폼을 효과적으로 관리할 수 있는 관리자 콘솔

**핵심 기능**:
- RBAC 기반 관리자 권한 시스템
- 사용자 관리 (전문가, 고객, 관리자)
- 주문 모니터링 및 개입 기능
- 서비스 카탈로그 관리 UI
- 정산 처리 및 수동 조정
- 통계 대시보드 (주문, 매출, 전문가 활동)
- 시스템 설정 및 관리

**출력물**:
- 완성된 백오피스 앱 (`sgsg-adm/`)
- Ant Design 기반 관리자 컴포넌트
- 전체 관리 기능 구현

### Phase 3: 전문가 웹앱 (3-4주) - Phase 1 완료 후
**목표**: 전문가가 현장에서 효율적으로 작업할 수 있는 모바일 퍼스트 웹앱

**핵심 기능**:
- 모바일 최적화 UI (하단 네비게이션)
- 마스터/서브 계정 전환 기능
- 주문 수락/거절 및 진행 상태 관리
- 일정 관리 (캘린더 뷰)
- 정산 내역 조회
- 프로필 및 포트폴리오 관리
- 실시간 알림 (SMS/KakaoTalk)

**출력물**:
- 완성된 전문가 웹앱 (`sgsg-exp/`)
- 모바일 퍼스트 Ant Design 컴포넌트
- 오프라인 지원 기능

### Phase 4: 소비자 웹 (향후) - Phase 1-3 완료 후
**목표**: 고객이 서비스를 검색하고 주문할 수 있는 웹 플랫폼

**핵심 기능**:
- 서비스 검색 및 필터링
- 주문 생성 및 결제
- 주문 추적
- 리뷰 시스템
- 프로필 관리

### 데이터베이스 구조 (Prisma 스키마)
기본 구조는 기존 `prisma/schema.prisma` 파일에 정의됨:
- **users** (사용자 기본) → customers, experts, admins 확장
- **service_categories** → **service_items** (3단계 계층)
- **orders** → order_notes, order_attachments, service_schedule
- **expert_service_mapping** (전문가-서비스 매핑)
- **payments** (결제 거래)
- **settlements** → settlement_details (정산 상세)
- **reviews** (리뷰 및 평점)
- **notifications** (시스템 알림)
- **audit_logs** (감사 로그)
