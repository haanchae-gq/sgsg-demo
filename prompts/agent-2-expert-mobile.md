## 📱 모바일 UX 패턴

### **네비게이션**
- **하단 탭바**: 주문(🏠), 일정(📅), 정산(💰), 내정보(👤)
- **상단 네비바**: 페이지 제목 + 뒤로가기 + 액션 버튼
- **알림**: 우상단 벨 아이콘 + 배지

### **제스처 지원**
- **풀투리프레쉬**: 데이터 새로고침
- **스와이프**: 주문 카드 액션 (수락/거절)
- **무한 스크롤**: 대용량 리스트
- **롱 프레스**: 컨텍스트 메뉴

### **알림 시스템**
- **실시간 푸시**: WebSocket 기반
- **로컬 알림**: 일정 미리 알림
- **뱃지 카운터**: 탭바에 새 주문 수 표시

## 💼 전문가 워크플로우

### **일반적인 사용 패턴**
1. **로그인** → 주문 대시보드 확인
2. **새 주문 알림** → 주문 상세 → 수락/거절  
3. **일정 확인** → 서비스 현장 이동
4. **서비스 진행** → 사진/메모 → 완료 처리
5. **정산 확인** → 수익 통계 확인

### **긴급 상황 플로우**
- 긴급 주문 → 즉시 알림 → 빠른 수락 → 실시간 위치 공유
