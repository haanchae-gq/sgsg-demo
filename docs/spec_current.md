## 아키텍처 특징
- 모노레포 구조: 백엔드(`sgsg-api`), 관리자 대시보드(`sgsg-adm`), 모바일 웹 앱(`sgsg-exp`) 분리
- CORS 프록시: Vite 개발 서버가 `/api` 요청을 백엔드로 프록시
- 환경 변수: 단일 `.env` 파일로 모든 서비스 공유
- Strict TypeScript: 모든 프로젝트에서 strict 모드 활성화

## 개발 서버 포트
- 백엔드 API: `http://localhost:4000`
- 관리자 대시보드: `http://localhost:3001`
- 모바일 웹 앱: `http://localhost:3002`

## 기술 스택 선택 배경
이 기술 스택은 현대적인 풀스택 웹 애플리케이션 개발에 최적화된 조합으로 선택되었습니다:
1. **Fastify**: Express보다 빠른 성능과 낮은 오버헤드
2. **Prisma**: TypeScript-first ORM으로 타입 안전성 보장
3. **React 18**: 최신 React 기능과 동시성 렌더링 지원
4. **Ant Design**: 엔터프라이즈급 UI 컴포넌트 라이브러리
5. **Vite**: 빠른 개발 서버와 빌드 최적화
6. **TypeScript**: 정적 타입 검사로 런타임 오류 감소

- **테스트 포인트**: 기존 라우트 핸들러, 미들웨어, 플러그인 등록 코드

**Prisma 5 → 7**

- **호환성 이슈**: 마이그레이션 파일 형식 변경, 일부 쿼리 API 변경, Prisma Client 생성 방식 변화
- **권장 버전**: prisma ^7.4.2, @prisma/client ^7.4.2
- **테스트 포인트**: Prisma Client 사용 모든 코드, 마이그레이션 스크립트, 시드 데이터

**Vite 4 → 7**
- **주요 변경사항**: Node.js 18 이상 요구, 빌드 성능 향상, 구성 파일 형식 변경, 플러그인 API 업데이트
