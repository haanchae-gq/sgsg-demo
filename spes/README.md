## 📋 문서 개요

이 폴더에는 쓱싹 홈케어 플랫폼의 새로운 요구사항을 반영한 스펙 문서가 포함되어 있습니다.

### 주요 변경사항
- **백엔드**: Supabase → PostgreSQL + Node.js (Express) + Socket.io
- **우선순위**: 전문가 웹앱 → 백오피스 → 사용자 웹
- **배포 전략**: 로컬 개발 → 클라우드 배포
- **추가 기능**: 마스터/서브 계정, 채팅, 3단계 카테고리, 가격 스냅샷

## 🎯 개발 우선순위 (최종)

### Phase 1: 백엔드 기초 (1-2주)
**목표**: 데이터베이스 및 기본 API 구축

**핵심 작업**:
- PostgreSQL 스키마 생성 (35개 테이블)
- Node.js + Express 프로젝트 설정
- Prisma ORM 설정
- JWT 인증 시스템
- Redis + Socket.io 설정
- 기본 API 엔드포인트

### Phase 2: 전문가 웹앱 (2-3주)
**목표**: 전문가가 주문을 관리하고 서비스를 제공할 수 있는 웹 애플리케이션

**핵심 기능**:
- 마스터/서브 계정 로그인
- 대시보드 (계정별 권한)
- 예약관리 (8개 상태 탭)
- 내부 배정 (마스터 → 서브)
- 채팅 (전문가 ↔ 고객)
- 캘린더 (팀원 필터)
- 팀원 관리 (마스터만)
- 정산 조회
- 프로필 관리

---

### Phase 3: 백오피스 (2-3주)
**목표**: 운영팀이 플랫폼을 관리할 수 있는 관리자 시스템

**핵심 기능**:
- 관리자 로그인
- 대시보드 (상세 지표)
- 사용자 관리 (전문가 팀 관리 포함)
- 주문 관리 및 모니터링
- 서비스 관리 (3단계 카테고리, 가격 스냅샷)
- 정산 관리 (상태 관리, 승인, 지급)
- 통계 및 리포트 (주문, 정산, Excel/PDF)

---

### Phase 3: 사용자 웹 (향후, 3-4주)
**목표**: 고객이 서비스를 검색하고 주문할 수 있는 웹 애플리케이션

**핵심 기능**:
- 고객 회원가입/로그인
- 서비스 검색 및 필터링
- 주문 생성 및 결제
- 주문 조회 및 취소
- 리뷰 작성
- 프로필 관리
