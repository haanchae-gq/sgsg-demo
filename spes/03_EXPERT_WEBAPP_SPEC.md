## 개요

### 목적
전문가가 주문을 효율적으로 관리하고 서비스를 제공할 수 있는 웹 애플리케이션

### 핵심 가치
- **효율성**: 빠른 주문 확인 및 처리
- **편의성**: 직관적인 UI/UX
- **투명성**: 명확한 정산 내역
- **신뢰성**: 안정적인 서비스 제공

### 주요 기능
1. 주문 관리 (조회, 수락/거절, 상태 업데이트)
2. 일정 관리 (캘린더 뷰, 일정 조정)
3. 정산 조회 (주간/월간 정산, 명세서 다운로드)
4. 프로필 관리 (개인정보, 서비스 정보, 계좌 정보)
5. 알림 (실시간 주문 알림, 일정 알림)

## 사용자 시나리오

### 시나리오 1: 신규 주문 수락
```
1. 전문가가 로그인
   ↓
2. 대시보드에서 신규 주문 알림 확인
   ↓
3. 주문 상세 정보 확인
   - 고객 정보
   - 서비스 내용
   - 주소 및 일정
   - 금액
   ↓
4. 일정 확인 후 수락 또는 거절
   ↓
5. 수락 시 일정 확정
   ↓
6. 고객에게 자동 알림 전송
```

### 시나리오 2: 서비스 진행 및 완료
```
1. 당일 일정 확인
   ↓
2. 서비스 시작 버튼 클릭
   ↓
3. 서비스 진행
   ↓
4. 추가 비용 발생 시 입력
   ↓
5. 완료 사진 업로드
   ↓
6. 서비스 완료 버튼 클릭
   ↓
7. 고객에게 잔금 결제 요청
```

### 시나리오 3: 정산 조회
```
1. 정산 메뉴 클릭
   ↓
2. 주간/월간 정산 내역 확인
   ↓
3. 정산 상세 내역 조회
   - 주문별 수익
   - 플랫폼 수수료
   - 순수익
   ↓
4. 정산 명세서 다운로드 (PDF)
```
